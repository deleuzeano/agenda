# 📅 Minha Agenda Pessoal

Agenda interativa para organizar rotina.


## ✨ Funcionalidades

- ✅ **Checkboxes** para marcar tarefas concluídas
- 📝 **Anotações** em cada horário do dia
- 📊 **Barra de progresso** mostrando o avanço das tarefas
- ⚡ **Modo Quarta Especial** (1x por mês)
- 🌙 **Destaque automático** do horário atual
- 📱 **Design responsivo** para celular e computador

## 🚀 Como acessar

**Link direto:** https://deleuzeano.github.io/agenda/

### Instalar no celular:

**iPhone (Safari):**
1. Abra o link acima
2. Toque no botão Compartilhar 📤
3. Escolha "Adicionar à Tela de Início"
4. Toque em "Adicionar"

**Android (Chrome):**
1. Abra o link acima
2. Toque nos 3 pontinhos ⋮
3. Escolha "Instalar app" ou "Adicionar à tela inicial"
4. Confirme a instalação

Agenda pessoal desenvolvida para organizar a rotina acadêmica de Mestrado e Direito.
